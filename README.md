# WeixinRobot
微信自动回复工具，通过网页微信接口，收到信息自动回复指定内容，可添加图灵机器人（人工智障）回复  
朋友撤回消息后自动通过文件传输助手发送撤回消息内容（支持文字、语音、图片、链接、名片等消息）  
可通过文件助手开关自动回复功能。  
# 如何使用：  
本程序基于python2.7版本，运行程序，扫描登录即可。
